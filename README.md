# dazi
[巧手打字通](https://laidazi.com)是一款专业的在线打字学习平台，以精准测试、高效练习、趣味提升为核心，为不同人群提供全方位的打字学习、练习与竞技服务，覆盖打字入门到高阶提升全流程，同时配套丰富的教程和社区功能。
## 一、核心学习与练习功能
平台功能板块划分清晰，覆盖多类型学习需求，适配不同基础用户：
1. [基础学习](https://laidazi.com/public/kb/kfp)：含[键盘指法](https://laidazi.com/public/kb/layout)、[入门课程](https://laidazi.com/public/kb/course/list)、[数字](https://laidazi.com/public/kb/exercise?type=number) / [拼音](https://laidazi.com/public/kb/course/list?actionType=keyboard_speak) / [英文](https://laidazi.com/public/kb/course/list?actionType=keyboard_english) / [中文练习](https://laidazi.com/public/kb/course/list?actionType=keyboard_works)、[音乐练习](https://laidazi.com/public/kb/course/list?actionType=keyboard_piano)等，从基础键位和输入法入手搭建打字能力；
2. 综合练习：提供[打字游戏](https://laidazi.com/public/kb/game/list)、[常规练习](https://laidazi.com/public/kb/rest/test)、[测试](https://laidazi.com/public/kb/to/test)、[比赛](https://laidazi.com/contest/public/welcome)、[考试](https://laidazi.com/public/test)等，还有[经典名著跟打](https://laidazi.com/famous/)，兼顾基础训练与实战应用；
3. 多语种拼读：支持英语、日语、德语、哈萨克语、印尼语拼读，还设有编程专项[拼读练习](https://laidazi.com/spelling/lang/en)；
4. 输入法专项：针对[五笔](https://laidazi.com/public/channel/wb)、[双拼](https://laidazi.com/public/kb/layout/sp)、[日语输入法](https://laidazi.com/public/channel/jp)打造专属练习模块，满足不同输入法使用需求。
## 二、社区与辅助功能
1. 社区相关：含[打字教程](https://laidazi.com/ct/list-all)、[打字排行](https://laidazi.com/public/rank/top/7)、[软件下载](https://laidazi.com/public/to?p=download)、[上网导航](https://laidazi.com/public/pilot)、[拼音转换](https://laidazi.com/public/translate/pinyin?pageSize=63)、[打字社区](https://laidazi.com/ct/agg-list-all)，方便用户交流学习、查看排名和获取配套资源；
2. 基础设置：支持打字声音设置，同时提供登录、注册功能，可进行技能认证解锁权益；
3. 推广福利：邀请好友注册、小红书分享打字心得，可领取 VIP 会员。
## 三、特色内容与服务
- 技能认证体系：设置青铜、白银、黄金等等级的[技能认证](https://laidazi.com/public/user-levels)，用户可通过练习考取对应等级，平台实时公示用户认证结果；
- 趣味竞技板块：拥有 50 + 款[趣味打字游戏](https://laidazi.com/public/game/all)（如星球大战、水果切切、打地鼠等），还有[在线打字比赛](https://laidazi.com/public/kb/race/list)，比赛文本涵盖中外经典诗文、小学课文、百家姓、千字文、生活哲理、编程内容等；
- 专项课程：打造零基础入门、拼音速成、数字键盘、唐诗宋词结合、英语专项、音乐节奏打字等[特色课程](https://laidazi.com/public/resource-list-all)，还将打字训练与传统文化、音乐创作、真实办公场景结合；
- 专业测试：提供[快速打字](https://laidazi.com/spelling/exam)、[键盘速度](https://laidazi.com/public/kb/rest/space)、[在线键盘](https://laidazi.com/public/kb/rest/keytest)、[反应速度](https://laidazi.com/public/kb/rest/reaction)等多维度测试，精准检测打字水平；
- 配套教程：除[打字教程](https://laidazi.com/ct/list-practice)外，还提供电脑基础（PPT/Word/Excel/Windows 操作）、搜索引擎使用等[教程](https://laidazi.com/ct/list-knowledge)，适配电脑小白。
## 四、用户支持与常见问题
- [常见问题](https://laidazi.com/ct/qa)解答：针对盲打练习、指法规则、输入法选择、打字速度提升、键盘使用等高频问题提供解答，覆盖练习全流程疑问；
- 专属[成长计划](https://laidazi.com/client/my/fission)：推出小红书打字成长计划，为用户提供学习打卡与提升路径。
## 五、用户评价
平台获得不同人群的认可，宝妈认为其适合孩子学习，大学生反馈打字速度提升 50%、学习工作效率提高，职场白领通过平台掌握盲打技巧，退休老干部借助平台学会双拼打字，整体评价偏向实用、高效、易上手。
